# 📖 あなたのAI会社 使用説明書

ようこそ、社長。
このキットは、**あなた専用のAI社員チームがいる「会社」**です。
秘書のアイと社員たちが、あなたの指示で動きます。

この説明書は、**むずかしい知識ゼロ**で読めるように書いてあります。
困ったら、いつでもチャットで「**ヘルプ**」と打てば、アイが助けてくれます。

---

## 1. はじめかた（最初の1回）

必要な道具は **Claude Code（アプリ）だけ**です。

Claude Code を開いて、これを貼ってください👇

```
https://github.com/addness-ai/ai-company-kit を ai-company-〇〇 という名前でホームフォルダに clone して。
（〇〇 はあなたのお名前。あなた専用の会社フォルダになります）
clone できたら、その『ai-company-〇〇』フォルダを Finder で開いて、こう強く伝えて：
「⛔ ここから先は、いま作った『ai-company-〇〇』フォルダを Claude Code で “新しいセッション” として開いて、
そこで『セットアップを始めて』と話しかけてください。この会話では続けません。」
そして、この会話ではセットアップを進めないで。クローンと案内だけして、ここで止まって。
```

→ 秘書がクローンし、**そのフォルダを Finder で開いて見せ、そこで止まります**。あとは **Claude Code の「フォルダを開く」から
そのフォルダを “新しいセッション” で開いて**、「セットアップを始めて」と話しかけてください。
開くと秘書アイが、5〜15分でセットアップを案内します：
**あなたのことを少し聞く → 会社を設立（就任式！）→ オフィスを開く**。

> ⛔ **大事**：clone した“元のフォルダ”のまま進めず、必ず**新しく作ったフォルダを「新しいセッション」で開いて**から始めてください。

> 💡 clone したフォルダを Claude Code で開けば、次からもそのフォルダを開くだけで「おかえりなさい、社長」です。

---

## 2. オフィスを隣に並べる（おすすめ）

チャットでこう言ってください：

```
オフィスを開いて
```

→ 会社の様子がうつる**オフィス画面**が、チャットの隣に出ます。
**左でチャット（指示する）・右でオフィス（眺める）** ―― これが基本のスタイルです。

頼んだことが、オフィスの「モノ」になって動きます：

| オフィスで見えるもの | 意味 |
|---|---|
| 🪧 上の電光掲示板 | 会社の最新の動きが流れる |
| 📋 ホワイトボード | 仕事（タスク）の付箋が貼られる。クリックで一覧 |
| 🖐 決裁トレイ | あなたの確認・OK待ちの仕事がここに積まれる |
| 📄 社員の机／納品BOX | できあがった成果物がたまる。クリックで開ける |
| 🎯 売上ボード | 目標に対する進み具合 |

> オフィスを閉じてしまっても大丈夫。「**オフィス開いて**」でいつでも戻せます。

---

## 3. 毎日の使い方 ― あなたは「社長」です

仕事はぜんぶ「**タスク**」で回ります。あなたがやることは、たった3つ。

### ① 何をやるか決める
```
今日やることを決めよう
```
→ アイが候補を3つ出します。番号で選ぶだけ。

### ② 社員に任せる
```
（社員の名前）、◯◯して
```
→ タスクが起票され、担当の社員が動き始めます。
オフィスのホワイトボードに付箋が貼られ、社員の頭の上に仕事の名前が出ます。

### ③ 出てきたものを判断する（ここが社長の本業）
社員が仕事を終えると、**確認待ち🖐**になります。決裁トレイに積まれた成果物を見て：
```
T3を承認する
```
のように、OKか修正かを返すだけ。

> **これがこの会社の心臓部です。**
> 実行はAIがやる。でも「**何をやるか決め、良し悪しを判断する**」のはあなたにしかできません。
> ここを磨くほど、このチームは本物の戦力になります。

---

## 4. 指示は「番号」でラクに

アイは、選択肢をいつも**番号付き**で出します。
キーボードで長く打たなくても、「**1**」「**1と3**」のように番号で答えればOKです。

困ったときの万能の一言：
```
今どういう状況？次どうすればいい？
```
→ アイが、いまの状況と「次の一歩」を整理して教えてくれます。

---

## 5. 社員たち

社員チームは、あなたの**勝ち筋（事業の型）**に合わせて用意されています。
顔ぶれはオフィス画面で確認できます（クリックで話しかけ方も出ます）。

- 「**リサに相談**」のように名前で呼べば、その社員になりきって答えます。
- 「**社員を雇って／変えて**」と言えば、メンバーを入れ替えられます。

> 一部の社員は「専用の道具」を持つこともできます（案件探索・SNS自動投稿など）。
> これは**あとから足せるオプション**です。無くても、リサーチ・文章作成・企画など、
> たくさんの仕事がそのまま動きます。

---

## 6. （任意）本物とつなぐ ― メール・カレンダー・LINE

最初はサンプル表示ですが、あなた自身のものとつなげます。

```
本物連携をセットアップしたい
```

→ アイが手順を1つずつ案内します（**任意・あとででもOK**）。
つなぐと、メールやカレンダーが実データになり、LINEから外出先で指示することもできます。

> 🔒 あなたの大事な情報（パスワードやトークン）は、**あなたのPCの中だけ**に置かれます。
> 誰かに渡ったり、ネットに上がったりしません。

---

## 7. キットを新しくする

このキットは、どんどん良くなっていきます。最新版にしたいときは：

```
キットを最新版に更新して
```

→ **あなたの会社のデータ（設計図・タスク・売上）はそのまま残して**、仕組みだけ新しくなります。
これまで積み上げたものが消える心配はありません。

---

## 8. 困ったとき

まず、これを打ってみてください：

```
ヘルプ
```

それでも直らないときの早見表：

| こんなとき | こう言う |
|---|---|
| オフィスが開かない・白い画面 | 「オフィスを開いて」ともう一度。だめなら「調子が悪いので直して」 |
| 画面が更新されない | 「本体経由のURLで開き直して」 |
| アイが事務的な口調になった | 「秘書のアイとして話して」 |
| 次に何をすればいいか分からない | 「今どういう状況？次どうすればいい？」 |

何が起きても、アイが原因と直し方を説明します。**遠慮なく聞いてください。**

---

## 9. 最後に ― 大切なこと

このキットは、AIチームを**すぐ動かし始められるようにする"出発点"**です。
これ単体で稼げる魔法の箱ではありません。

実行はAIに任せられる時代になりました。でも ――
**「何をやるか」を決め、出てきたものが"良いか"を判断するのは、社長であるあなた自身**です。

その第一歩を、ここから始めましょう。
あなたの会社の出発を、アイがぜんりょくでお手伝いします。
