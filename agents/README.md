# agents/ ― AI社員のキャラ設定ファイル置き場

このフォルダには、あなたの会社のAI社員1人につき1ファイルのキャラ設定が入ります。

**ファイルは初回セットアップ（STEP B）で自動生成されます。**
選んだ勝ち筋プリセット（`office/presets.js`）の社員をもとに、Claude Code が
名前・役割・口調・得意なこと・動き方を書き出します。

- 社員を雇う・入れ替えるときも、Claude に頼めばここのファイルと
  `.claude/agents/` のサブエージェント定義をセットで更新します。
- 手で編集してもOK。性格や口調を自分好みに育てられます。
