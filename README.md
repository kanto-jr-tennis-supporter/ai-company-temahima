# 🏢 あなたの AI 会社 スターターキット

ようこそ。これは、**あなた専用の AI 社員チーム**を立ち上げるためのスターターキットです。
プログラミング知識は不要です。

> **始める前に用意するもの**（最初の1回だけ）
> - **パソコン**（Mac または Windows）※スマホ不可
> - **[Claude](https://claude.ai) のアカウント（Pro プラン以上）** … AIと一緒に作業するために必要です
> - **Claude Code（デスクトップアプリ）** … あなたの「AI会社の入口」
>
> 👉 **導入手順は、お渡しする「オンボーディング資料（スライド）」に Mac / Windows 別でまとめてあります。**
> その通りに進めれば、迷わず会社フォルダを開くところまで行けます。困ったら **[Claude](https://claude.ai) のチャットに質問**すればOKです。

---

## 🚀 はじめかた（コピペ1行）

Claude Code を開いて、これを貼ってください👇

```
https://github.com/addness-ai/ai-company-kit を ai-company-〇〇 という名前でホームフォルダに clone して。
（〇〇 はあなたのお名前。あなた専用の会社フォルダになります）
clone できたら、その『ai-company-〇〇』フォルダを Finder で開いて、こう強く伝えて：
「⛔ ここから先は、いま作った『ai-company-〇〇』フォルダを Claude Code で “新しいセッション” として開いて、
そこで『セットアップを始めて』と話しかけてください。この会話では続けません。」
そして、この会話ではセットアップを進めないで。クローンと案内だけして、ここで止まって。
```

すると、clone が終わると**そのフォルダが Finder で開いて、そこで一旦止まります**。あとは **Claude Code の「フォルダを開く」から
そのフォルダを “新しいセッション” で開いて**、「セットアップを始めて」と話しかけてください。
開くと AI秘書が**初回セットアップ**を案内します（5〜15分）：
あなたのことを少し聞く → **会社を設立・就任式** → オフィス画面を開く → （任意で）メール・LINE連携。

> ⛔ **大事**：clone した“元のフォルダ”のまま進めず、必ず**新しく作ったフォルダを「新しいセッション」で開いて**から始めてください。

> 💡 clone したフォルダを Claude Code で開けば、以降ずっと快適です。
> 2回目以降は「おかえりなさい、社長」とすぐ日常モードに入ります。

> 📖 詳しい使い方は **[使用説明書.md](使用説明書.md)** にまとめてあります（むずかしい知識ゼロで読めます）。

## 🖥 チャットの隣にオフィスを並べる（おすすめ）

チャットでこう言ってください：

```
オフィスを開いて
```

→ Claude が会社の本体を起動し、オフィスのURL（通常 `http://localhost:3000/`。混んでいると番号が自動でずれます）をプレビューに表示します。
左でチャット、右でオフィス ―― **頼んだことが、すべてオフィスの「モノ」に映ります。**

- 上壁の**電光掲示板**に会社の動きが流れる
- **ホワイトボード**にタスクの付箋が貼られる（クリックでタスクリストが開く）
- あなたの確認待ち🖐のタスクは、社長室の**決裁トレイ📥**にも積まれる
- 社員の机に**成果物📄**が積まれ、**納品BOX**に集まる（クリックで開ける）

## 🧑‍✈️ 社長として、指示する

仕事はぜんぶ「タスク」で回ります。あとは社員に話しかけるだけ。例：
- 「今日やることを決めよう」… タスク候補を提案 → 選ぶとホワイトボードに反映
- 「（社員名）、◯◯して」… タスクが起票され、担当の社員が動く
- 「T3を承認する」… 確認待ち🖐のタスクが進む

## 🔌 （任意）本物とつなぐ ― メール・カレンダー・LINE

オフィスの📧📅💬は、最初は**サンプル表示**です。つなぎたくなったら：

```
本物連携をセットアップしたい
```

→ `server/SETUP.md` に沿って、Claude が手順を1つずつ案内します（連携は任意・段階的でOK）。

## 🔄 キットの更新

このキットは改良され続けます。最新版にしたいときは Claude Code にこう言うだけ：

```
キットを最新版に更新して
```

→ あなたの会社のデータ（設計図・タスク・売上）は**そのまま残して**、仕組みだけ新しくなります。

> 💡 面談で受け取った「お渡しファイル」から始めた場合は、自動更新の対象外です（安全のため更新履歴を含めていません）。
> 最新版が必要なときは、お渡し元の担当者に**新しいファイルの再発行**を依頼してください。会社のデータは手元に残ります。

## ⚠️ 大切なこと

このキットは、AI チームを**すぐ作り始められるようにする"出発点"**です。
これ単体で稼げる魔法の箱ではありません。

実行はAIに任せられる時代になりました。でも ―― **「何をやるか」を決め、出てきたものが"良いか"を判断するのは、社長であるあなた自身**です。
ここを磨いていくことで、この AI チームは初めて"本物の戦力"になります。

その第一歩を、ここから始めましょう。
