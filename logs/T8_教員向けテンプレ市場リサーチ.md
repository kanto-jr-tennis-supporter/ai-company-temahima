# T8：教員向けテンプレ商品の市場リサーチ

- 調査日：2026-07-07
- 担当：🔍 リサ（案件リサーチャー）
- 目的：社長のGAS×スプレッドシートスキルで作れる「教員向けテンプレ・ツール商品」を見極め、月1.5〜3万円の継続収入源の種を特定する

---

## 1. 教員は何に困っているか（需要調査）

### 公的データが示す「事務作業のペイン」

- 文科省「教員勤務実態調査（令和4年度）」：1日平均在校時間は**小学校教諭で10時間、中学校教諭で11時間超**。
  https://www.mext.go.jp/content/20240404-mxt_zaimu01-100003067-2.pdf
- みんなの教育技術（小学館）の現役教員約5,200人アンケート：**約9割が隠れ残業、平均11時間勤務、3人に2人が休憩取れず**。
  https://kyoiku.sho.jp/394047/
- 栃木県教委の多忙感調査：負担感の上位は**「成績処理」「授業準備」「事務・報告書作成」（いずれも25%超）**。児童生徒に直接関わらないデスクワークほど「忙しさ」を感じる傾向。
- OECD調査（日経報道）：日本の教員は**事務作業と保護者対応のストレス**が国際平均より高く、前回調査より悪化。部活動も勤務時間を押し上げる主因。
  https://www.nikkei.com/article/DGXZQOUD027V70S5A001C2000000/
- 東洋経済のTwitter教員アカ座談会：部活動指導・膨大な事務作業で「現場は疲弊」が共通の声。
  https://toyokeizai.net/articles/-/639466

### テーマ別の温度感（調査から見えた序列）

| テーマ | 需要の強さ | 根拠・備考 |
|---|---|---|
| 成績処理・テスト集計 | ★★★ | 負担感調査で常に上位。デジタル採点導入校で採点時間50%減の事例（金沢市立清泉中）など「効率化すれば効く」実証多数 |
| 部活動管理（出欠・会計・大会） | ★★★ | OECD調査でも勤務時間押し上げ要因。地域移行の過渡期で管理ツール需要が顕在化（総務省DXナビでも特集） |
| 学級事務の一括作成（座席表・名簿・名札） | ★★★ | 実際にnoteで売れている（後述コギト氏）。「4月の事務ラッシュ」という明確な購買タイミングがある |
| 行事の集計（体育祭・運動会の得点） | ★★☆ | 「タブレット×スプシでリアルタイム得点反映」のnote記事があり関心は高いが、**商品としてはほぼ空白地帯** |
| 授業時数・週案管理 | ★★☆ | 教務主任向け。noteの八重山氏が有料記事で展開中＝売れるテーマの証拠 |
| 保護者連絡・アンケート集計 | ★★☆ | 約9割の小中学校でフォーム化が進行中（文科省）。「フォームは作れたが集計・分析ができない」層が残る |
| 出欠管理（授業・学級） | ★★☆ | ココナラに出品例あり。ただし校務支援システムと被る学校も |
| 通知表所見 | ★☆☆ | 需要は大きいが、小学館が**無料の所見文例DB**を公開済み＋個人情報×AIの規約リスクあり |

## 2. 実際に何が売れているか（プラットフォーム別）

### note ＝ 教員向けテンプレ販売の主戦場（実証済み）

**実例1：コギト氏（音楽授業の教材つくる人）** https://note.com/cogito1979/n/nf65fb91b6e83
- 「教員の4月事務効率化スプレッドシート」：**¥2,980**
  - 名簿を1回貼るだけで6種類を一括生成（座席表2種・出席チェック表・提出物チェック表・名札・掲示用名簿）
- 「年間指導計画作成ツール」：**¥1,980**
- 2本セットのバンドル：**¥3,980**（980円お得の抱き合わせ設計）
- ポートフォリオ全体で**5,000ダウンロード超**。Excel版・特別支援学校版など横展開もしている

**実例2：八重山智也氏（仕組みで校務をラクにする人）** https://note.com/yamahachi_sensei
- 「ICT活用で学校の仕事を効率化」マガジン**109本**。GAS×スプシの校務ネタで継続発信
- 学校日誌のGASデジタル化、時数管理、生徒指導記録テンプレ等
- **書籍化に到達**（『業務効率を200%アップ！AI&ICT活用の最強仕事術』テンプレ約40個付き）＝このジャンルは出版社が本を出すレベルの市場

**noteの価格相場**（有料記事の一般相場）：テンプレ・専門ノウハウは**800〜1,500円**、ツール一式・PDF付きは**2,000円台**、業界特化の本格教材は**5,000〜10,000円**。手数料は事務手数料5%＋プラットフォーム利用料10%（≒手取り約85%）。
https://www.sales-dx.jp/blog/note-informative-notes

### ココナラ ＝ 「テンプレ販売」より「カスタム受託」の場

- GAS作成・制作カテゴリは**実績3,000件超**の市場。https://coconala.com/categories/230/737
- 相場：GAS修正・小規模作成が**1,000円〜**、自動化一式は見積もり制（数千〜数万円）
- 出欠管理の出品例：「EXCEL苦手でも大丈夫！生徒の出欠管理できます」https://coconala.com/services/3218691
- **重要な発見：「教員向け・学校向け」に特化したGAS出品はほぼ空白**。一般向け自動化ばかりで、「先生の言葉で先生の業務を語れる出品者」がいない

### ストアカ ＝ GAS講座は一般向けが中心

- GAS個別レッスン等はあるが（https://www.street-academy.com/myclass/136871 ）、**教員特化のGAS・校務効率化講座は薄い**。アビバのGAS講座は年間13,000人受講＝「GASを学びたい」市場自体は大きい

### 競合として注意すべき無料勢

- 座席表・名簿・出欠表の**単品テンプレは無料が豊富**（Microsoft公式、eTeachers、planplan等）→ 単品では売れない。コギト氏のように「一括生成・時短の仕組み」まで作り込んで初めて有料になる
- 新体力テスト集計は福岡県が無料ソフト配布（小学校中心）
- 部活連絡はBAND・サークルスクエア等の無料アプリが強い。ただし**保護者全員にアプリを入れさせるハードル**があり、「顧問1人で完結するスプシ管理」は別の土俵
- 個人事業主向け売上管理スプシは**無料配布（リスト取り）が主流**で有料販売は激戦 → 横展開先としては「テンプレ販売」より「カスタム受託」向き

## 3. 社長に作れるか判定（GAS＋スプシ＋Claude活用）

社長の実績：テレアポシートの自動転記・集計／入札申請書アプリ／部活動出欠確認シート

| テーマ | 判定 | 理由 |
|---|---|---|
| 部活動管理（出欠・会計・大会・連絡） | **◎すぐ作れる** | 部活動出欠確認シートを**既に作った実績あり**。会計・大会エントリーは転記・集計の応用 |
| 体育祭・球技大会の得点集計 | **◎すぐ作れる** | テレアポシートの自動集計と同じ構造（入力→自動集計→表示）。保健体育＋行事とりまとめ経験で要件が全部頭に入っている |
| 4月学級事務パック（座席表・名札等） | **◎すぐ作れる** | 名簿からの自動生成＝転記スキルそのもの。ただしコギト氏と真っ向競合 |
| アンケート集計（フォーム→自動集計→グラフ） | **◎すぐ作れる** | フォーム連携＋集計はGASの得意技。行事後アンケート・学校評価に応用可 |
| 新体力テスト集計・分析（中高版） | **○数日** | 集計は容易。学年別・男女別のT得点/評価基準の実装に少し設計が要る。保体の専門知識が差別化になる |
| 授業時数・週案管理 | **○数日** | カレンダー連動のロジックがやや複雑。八重山氏が既に強いテーマ |
| 成績処理（観点別評価対応） | **○数日** | 集計自体は容易だが、学校ごとの評価方式の違いを吸収する設計が必要 |
| 通知表所見支援（AI活用） | **△要注意** | 技術的には可能だが、生徒個人情報×外部AIは自治体規約リスク。商品化は慎重に |
| 保護者連絡アプリ的なもの | **△非推奨** | 無料アプリ（BAND・LINEスクール連絡帳等）と正面衝突。作らない方がいい |

## 4. おすすめ商品候補トップ3

### 🥇 第1位：部活動まるごと管理キット（出欠・会計・大会エントリー・連絡テンプレ）

- **想定価格**：note ¥1,980〜2,980（会計・大会管理を足した完全版は¥3,980バンドル）
- **販売場所**：note（本命）＋ ココナラで「あなたの部活仕様にカスタマイズ」を¥5,000〜で併売
- **作る手間**：◎ 既存の部活動出欠確認シートが土台。1〜2週間で商品化可能
- **おすすめ理由**：
  1. 部活顧問は中高教員のほぼ全員が経験する巨大ペインで、OECD調査でも勤務時間押し上げの主因
  2. 無料アプリは「保護者全員にアプリを入れさせる」壁がある。**顧問1人で今日から使えるスプシ**は別の土俵で戦える
  3. 部活動の地域移行の過渡期で管理ニーズが顕在化中（総務省DXナビでも特集される旬テーマ）
  4. 現役の部活顧問が作った、という**当事者の説得力**が最強の販売文になる

### 🥈 第2位：体育祭・球技大会 リアルタイム得点集計キット

- **想定価格**：note ¥2,980
- **販売場所**：note（体育祭シーズンの4〜5月・8〜9月前に投稿を仕込む）
- **作る手間**：◎ 入力→自動集計→速報表示はテレアポシートと同じ構造。1週間程度
- **おすすめ理由**：
  1. **保健体育×行事とりまとめ経験のど真ん中**。要件定義が一切不要で、社長にしか書けない販売ページが書ける
  2. 関心を示すnote記事はあるのに、**商品としてはほぼ空白地帯**（コギト氏も八重山氏も未参入）
  3. タブレット入力→本部で自動集計→スクリーン速報、という「映える」デモがSNSで拡散されやすい
  4. 球技大会・合唱祭など行事シリーズへの横展開が容易

### 🥉 第3位：行事・学校評価アンケート自動集計キット（フォーム→集計→報告用グラフ一発）

- **想定価格**：note ¥1,480〜1,980
- **販売場所**：note ＋ ココナラで「フォーム設計ごと代行」受託（¥5,000〜10,000）への入口商品に
- **作る手間**：◎ フォーム連携＋自動集計＋グラフ生成。1週間程度
- **おすすめ理由**：
  1. 約9割の学校でフォーム化が進んだ結果、**「集めたはいいが集計・報告書化が手作業」**という次のペインが生まれている
  2. 体育祭アンケート・学校評価・保護者アンケートと年中使えて、買い手の職種を選ばない（担任・分掌・管理職）
  3. 分掌（教務・研究部）向けなので「学校の予算で買う／校内に広める」展開の芽がある

### 戦略メモ（3つに共通）

- **note中心＋無料記事で集客→有料テンプレ→ココナラでカスタム受託**の3段構え。コギト氏（単発¥2,980×5,000DL規模）と八重山氏（109本→書籍化）が、この市場で個人教員が稼げることを実証済み
- 単品テンプレ（座席表だけ等）は無料競合に潰されるので出さない。**「仕組みごと・一式で」**が有料化の条件
- note手取りは約85%。月1.5〜3万円なら**¥1,980×月8〜18本**。部活キット＋体育祭キットの2本柱＋シーズン記事で現実的に届く水準
- ココナラの「教員向けGAS」はほぼ空白。テンプレが売れ始めたら「学校の困りごと専門の自動化屋」として出品すれば、月5〜10万円ゴールへの単価アップ経路になる
